Scansion API

The public API is versioned, supports single-poem, batch, and streamed line-by-line scans, and powers the embeddable widget below. Responses include the x-scansion-api-version header and an apiVersion field in the JSON body.

Endpoint
POST /api/scan
Version
v1
Payload limits

Single poem: 24,000 characters. Batch: 10 poems / 120,000 characters.

Public usage guidance

Soft limit: keep automated traffic below roughly 60 requests per minute per client.

Single scan

{
  "poem": "Shall I compare thee to a summer's day?",
  "profile": "early_modern",
  "includeSummary": false
}

The response returns poem-level metadata, line scans, rhyme groups, form detection, diagnostics, and the active pronunciation profile.

Streamed scan

{
  "poem": "Because I could not stop for Death -\nHe kindly stopped for me -",
  "stream": true,
  "includeSummary": false
}

Streaming responses use application/x-ndjson and emit line, complete, and optional summary events in order.

Batch scan

{
  "profile": "modern",
  "poems": [
    { "id": "p1", "title": "Poem one", "poem": "..." },
    { "id": "p2", "title": "Poem two", "poem": "...", "profile": "hymn" }
  ]
}

Batch mode returns per-poem summaries using the same core engine as the single scan path.

Overrides

{
  "poem": "At Recess - in the Ring -",
  "overrides": {
    "tokens": {
      "0:1": {
        "stressPattern": "su",
        "pronunciation": "R IH0 S EH1 S",
        "label": "noun reading"
      }
    }
  }
}

Overrides can target a specific token by lineIndex:tokenIndex or a normalized word. The frontend uses token overrides so readers can correct one line without rewriting the whole poem.

Embeddable widget

Embed the widget with a custom poem, profile, or endpoint. The widget fetches scans from the same versioned API and can be initialized manually through window.ScansionWidget.renderScansionWidget().

<script type="module" src="https://poetryscansion.netlify.app/widget.js"></script>
<div
  data-scansion-widget
  data-profile="modern"
  data-max-lines="4"
  data-poem="Amazing grace! How sweet the sound
That saved a wretch like me!
I once was lost, but now am found,
Was blind, but now I see."
></div>
